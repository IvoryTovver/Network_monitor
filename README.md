# Network Monitor

A Python-based application for monitoring system and network performance.

## Features
- Displays CPU, memory, and disk usage.
- Lists network interfaces and their details.
- Monitors network traffic (bytes sent/received).

## Requirements
- Python 3.x
- `psutil` library
